# Урок 1. Первый сервер на Flask

## Что такое Flask

Flask — это лёгкий веб-фреймворк на Python, который позволяет быстро создавать веб-приложения.

---

## Установка

```bash
pip install flask
```

---

## Первый сервер

Создадим файл `app.py`:

```python
from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "Главная страница"

@app.route('/about')
def about():
    return "О нас"

if __name__ == '__main__':
    app.run(debug=True)
```

---

## Запуск

```bash
python app.py
```

После запуска сервер будет доступен по адресу:

```
http://127.0.0.1:5000/
```

---

## Задание

Создать сайт из 3 страниц:

* Главная (`/`)
* О себе (`/about`)
* Контакты (`/contact`)

Каждая страница должна возвращать осмысленный текст.
